First time setup

You need a GitHub Personal Access Token to use this panel. Here's how to get one:

  1. Go to github.com → Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens
  2. Click Generate new token
  3. Give it a name like "PP Archive"
  4. Under Repository access, select only your archive repo
  5. Under Permissions → Repository permissions, set Contents to Read and write
  6. Generate and paste the token below

Your token is stored only in your browser's memory and is never saved anywhere.
